# Student Grade Calculator

## Project Overview

The Student Grade Calculator is a Python program that accepts a student's name and marks, calculates the corresponding grade, and displays a motivational message based on performance.

## Features

* Accepts student name and marks as input
* Validates marks between 0 and 100
* Uses functions for grade calculation
* Uses if-elif-else statements
* Displays grades and encouraging messages
* Handles invalid input

## Grade Criteria

| Marks Range | Grade |
| ----------- | ----- |
| 90 - 100    | A     |
| 80 - 89     | B     |
| 70 - 79     | C     |
| 60 - 69     | D     |
| Below 60    | F     |
