# Personal Introduction Program

## 📌 Description
This is a simple Python program that takes user input and displays a personalized welcome message.

## 🛠️ Features
- Uses `input()` to collect user information
- Stores data in variables
- Uses `print()` and f-strings
- Friendly and welcoming output

## ▶️ How to Run

1. Install Python
2. Open terminal or command prompt
3. Run the program using:

```bash
python personal_intro.py
